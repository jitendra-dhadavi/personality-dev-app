Put your reel videos here.

This project references videos via `src/data/reelsData.json`, using public paths like:

- `/videos/video1.mp4`
- `/videos/video2.mp4`
- `/videos/video3.mp4`

So you should create files at:

- `public/videos/video1.mp4`
- `public/videos/video2.mp4`
- `public/videos/video3.mp4`
